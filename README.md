# 口腔管家管理系统 java840

口腔管家管理系统是一款基于SpringBoot和Vue的前后端分离系统。

## 技术栈

- 前端：Vue
- 后端：SpringBoot
- 数据库：MySQL
- 开发工具：Idea，vscode
- 插件：mybatis-plus，element

## 功能模块

### 系统角色

- 普通用户
- 超级管理员

### 主要功能

- **牙齿保健产品管理**：包含产品信息的增删改查等功能。
- **复查提醒管理**：管理用户的复查提醒和通知。
- **预约管理**：处理和安排用户的预约请求。
- **药品信息管理**：维护药品的相关信息。
- **病例信息管理**：管理用户的病例信息。
- **公告管理**：发布和更新系统公告。
- **用户管理**：对系统用户进行管理。
- **个人中心**：用户可查看和编辑个人信息。
- **个人后台**：提供个人账户相关的操作管理。

## 运行环境

- 前端：支持现代浏览器
- 后端：Java 8及以上版本
